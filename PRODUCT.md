# Growth Horizon Product Notes

## Purpose

Growth Horizon is a static student blog that preserves the idea of an old high school Technical Writing project while presenting it with portfolio-ready frontend quality.

## Audience

- Portfolio reviewers evaluating frontend craft and project judgment
- Students or readers interested in personal growth reflections
- The original project owner, who needs a clean project that is easy to explain

## Site Goals

- Present the original blog concept professionally
- Make posts easy to browse, search, filter, and read
- Show practical modernization work without adding unnecessary complexity
- Keep setup and deployment simple for static hosting

## Content Structure

- `index.html`: branded landing page, featured story, and blog library
- `main-post.html`: featured article
- `post1.html` through `post6.html`: individual student reflections
- `about.html`: project background and modernization context
- `members.html`: original student contributors

## Scope Limits

This project is a static frontend site. It does not include a backend, database, CMS, authentication, payments, analytics, external APIs, or paid services.

## What This Project Is

- A polished portfolio version of an older school blog
- A beginner-to-intermediate friendly static website
- A practical frontend refactor and UI/UX modernization case study

## What This Project Is Not

- A production CMS
- A full-stack blog platform
- A monetized or authenticated application
- A complete rewrite in a new framework
