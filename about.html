Project background

A school blog rebuilt with portfolio standards.

Growth Horizon began as a high school Technical Writing project by Group 2. The original goal was to collect student essays about personal development and present them as a simple blog website.

This modernized version keeps that purpose intact while improving the design, structure, responsiveness, accessibility, and overall presentation. It is still a beginner-friendly static website, but it now reads like an intentional portfolio project instead of an unfinished class assignment.

The content remains centered on reflection, resilience, and growth. The code now uses semantic HTML, a consolidated stylesheet, and a small amount of vanilla JavaScript for navigation and blog filtering.

Project focus

The update prioritizes maintainable static files, readable blog layouts, responsive cards, accessible navigation, and professional copy. No backend, login system, database, or paid service is required.
